# REST API Laravel Training Freelance

## Tech Stack

-   Laravel 8
-   MySQL 8.0

## Installation

1. Clone this repository
2. `composer install`
3. `cp .env.example .env`
4. `php artisan key:generate`
6. `php artisan serve`
